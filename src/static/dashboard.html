    Checklist Red Flags

    • Ada tautan pendek / domain mencurigakan
    • Bahasa mendesak atau ancaman penalti
    • Meminta OTP, PIN, password, atau data KTP
    • Iming-iming hadiah atau cashback berlebihan

    Respon 3 Langkah

    1. Hentikan interaksi. Jangan klik link dan jangan balas.
    2. Verifikasi lewat kanal resmi (nomor CS resmi, aplikasi resmi, situs resmi).
    3. Laporkan dan dokumentasikan bukti pesan untuk edukasi keluarga/komunitas.
